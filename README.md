# Gates

A collection of useful gates for use with Wiremod.

## The included gates

* Encoding
	* Base-2 Encode (Binary)
	* Base-8 Encode (Octal)
	* Base-10 Encode (Decimal)
	* Base-16 Encode (Hex)
	* Base-32 Encode
	* Base-64 Encode
	* URL Encode

* Decoding
	* Base-2 Decode (Binary)
	* Base-8 Decode (Octal)
	* Base-10 Decode (Decimal)
	* Base-16 Decode (Hex)
	* Base-32 Decode
	* Base-64 Decode
	* URL Decode

* Bitwise
	* Or (Many)
	* And (Many)
